# version

The `version` command prints the OpenBao version:

```shell-session
$ openbao version
OpenBao v1.2.3
```

It can also be printed by adding the flags `--version` or `-v` to the `openbao` command:

```shell-session
$ openbao -v
OpenBao v1.2.3, built 2022-05-03T08:34:11Z
```
